# Vigenère Cipher

This program accepts a user-input message and outputs back an encrypted and decrypted message using the Vigenère cipher method. 
## Installation



```bash
pip install foobar
```

## Usage

```python
import foobar

# returns 'words'
foobar.pluralize('word')

# returns 'geese'
foobar.pluralize('goose')

# returns 'phenomenon'
foobar.singularize('phenomena')
```
